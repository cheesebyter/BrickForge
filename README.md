# BrickForge

**BrickForge** is a local-first AI-assisted system for generating buildable brick-compatible models and simple building instructions.

> This is **MVP 0** – a technical proof of concept. It is not a finished product.

---

## Purpose of MVP 0

MVP 0 answers one core question:

> Can BrickForge locally generate a simplified brick model from a text description and export it as structured files?

The MVP 0 chain:

```
Prompt
  -> local Ollama analysis
  -> minimal model briefing
  -> simple template
  -> BrickGraph
  -> minimal validation
  -> LDraw/MPD file
  -> CSV parts list
  -> Markdown instructions
```

MVP 0 deliberately omits: full agent architecture, PostgreSQL, production API, complex validation, multiple templates, PDF export, cloud/external AI.

---

## Prerequisites

- [.NET 9 SDK](https://dotnet.microsoft.com/download)
- [Ollama](https://ollama.com/) running locally on `http://localhost:11434`
- Recommended model: `llama3.1:8b`

Pull the model before first use:

```bash
ollama pull llama3.1:8b
```

---

## Quick Start

### Check Ollama availability

```bash
dotnet run --project src/BrickForge.Cli -- health
```

### Generate a brick model

```bash
dotnet run --project src/BrickForge.Cli -- generate "Erstelle eine kleine schwarze Kaffeemaschine mit silbernem Frontpanel und einer Tasse."
```

Output files are written to `data/outputs/<jobId>/`.

## Project Structure

```
BrickForge/
  src/
    BrickForge.Cli/          CLI entry point
    BrickForge.Core/         Shared domain abstractions and result types
    BrickForge.Ai/           Ollama client and AI prompt handling
    BrickForge.BrickGraph/   BrickGraph domain model and generation
    BrickForge.Export/       LDraw/MPD, CSV and Markdown export
  tests/
    BrickForge.Core.Tests/
    BrickForge.BrickGraph.Tests/
    BrickForge.Export.Tests/
  data/
    outputs/                 Generated model output (per job)
    samples/                 Sample prompts
    parts/                   Supported parts catalogue
  docs/
    tickets/
    specs/
    ai/
```
